---
title: Installation
description: Install Minions Oss-ideas packages
---

## TypeScript / Node.js

```bash
npm install @minions-oss-ideas/sdk minions-sdk
```

## Python

```bash
pip install minions-oss-ideas
```

## CLI

```bash
npm install -g @minions-oss-ideas/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
