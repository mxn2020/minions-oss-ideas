# Minions Oss-ideas

Idea brainstorms, feasibility scores, tech stack picks, and market gaps

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-oss-ideas/sdk`) — Core library for Node.js
- **Python SDK** (`minions-oss-ideas`) — Python bindings
- **CLI Tool** (`@minions-oss-ideas/cli`) — Command-line interface
- **Web App** — Interactive playground at [oss-ideas.minions.wtf](https://oss-ideas.minions.wtf)
